# How to clone the project
1. Clone the repo to local computer
2. CD into flipgrid-challenge folder
3. run ```npm install```
4. Start the server by running `npm start`

# Tech Stack
- ReactJS
- CSS
- react-router
- react-router-dom
- yup (for validation)
- formik (async form validation)
